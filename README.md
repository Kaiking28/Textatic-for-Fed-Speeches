# Textatic for Fed Speeches

### A Natural Language Processing toolkit for analyzing language trends and sentiment patterns in Federal Reserve speeches.

---

## Overview

This project applies NLP techniques to Federal Reserve speeches to uncover linguistic trends, sentiment shifts, and communication patterns that may signal policy changes or economic outlooks. By analyzing the language used by Fed officials over time, we can identify how messaging evolves during different economic conditions.

## Features

### Speech Data Collection
Automated parsing of Federal Reserve speeches from official sources

### Text Preprocessing
Cleaning and normalization of speech transcripts

### Sentiment Analysis
Track emotional tone and confidence levels in Fed communications

### Trend Analysis
Identify recurring themes and language patterns across time periods

### Visualization
Generate insights through charts and graphs showing linguistic trends
